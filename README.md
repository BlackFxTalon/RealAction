<ul>

## Steps to start developing your new project with this template:

## clone project

<code>git clone https://github.com/BlackFxTalon/RealAction</code>

## install dependencies

<code>npm i</code>

## launch development

<code>npm run dev</code>

## launch production

<code>npm run build</code>
